Frequently Asked Questions

What file format is accepted?

We accept CSV (Comma Separated Values) files. Make sure your file has at least one column containing text to analyze.

How does the sentiment analysis work?

We use natural language processing to analyze text sentiment. Each piece of text receives a score from -1 (very negative) to 1 (very positive). These scores help predict public opinion trends.

Is my data secure?

Yes, all uploads are processed locally on the server and are not shared with third parties. Files are given unique identifiers to prevent conflicts.
